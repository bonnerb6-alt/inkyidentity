# Frontend Design Skill

When invoked, redesign the target page with these principles:

- **Visual-first**: Lead with imagery and graphics, not walls of text
- **Minimal copy**: Short punchy headlines, 1-2 line descriptions max
- **Rich visuals**: SVG illustrations, gradients, mockups, icons — no placeholder text blocks
- **Whitespace**: Breathe. Don't pack everything together
- **Dark aesthetic**: Deep blacks, purple accents (#7c3aed), glows
- **Motion**: CSS animations — floats, fades, pulses, scan lines
- **Sections should feel distinct**: vary backgrounds, use full-width imagery
- **Mobile-aware**: Use clamp(), flexWrap, auto-fit grids
